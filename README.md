# Discord Bot 만들기

> `discord.js`로 만든 간단한 디스코드 봇

## 실행방법

> Version  
> node.js: >=18.12.0  
> @discordjs/core: 0.6.0  
> @notionhq/client: 2.2.3

```bash
$ npm install

$ touch .env

# register your commands to the guild specified
$ npm run commands

# running you bot
$ npm run start
```
- [.env](https://discord.com/developers/applications/)
```
DISCORD_TOKEN=
CLIENT_ID=
GUILD_ID=
NOTION_TOKEN=
```

# Discord Bot 만들기

> `discord.js`로 만든 간단한 디스코드 봇

## 실행방법

> 버전  
> node: 18.12.1  
> discord.js: 14.7.1

```bash
$ npm install

$ touch .env

# register your commands to the guild specified
$ npm run commands

# running you bot
$ npm run start
```
- [.env](https://discord.com/developers/applications/)
```
DISCORD_TOKEN=
CLIENT_ID=
GUILD_ID=
NOTION_TOKEN=
```
